ONCURVE

ONCURVE is a tool to analyze Magic The Gathering decks

A common metric for defining if a deck is staying 'on curve' is if it is able to use all available mana for the first 4 turns. With ONCURVE - you can run many experiments to understand the likelihood you can achieve this.

Disclaimers/Assumptions:

  • Decks are mono colored
  • All lands are basic lands of the same color
  • Mana dorks or other tap effects are not taken in to account
